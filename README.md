# gbatemp-matias3ds-cheats

A curated collection of Nintendo Switch cheat files, regularly updated from GBAtemp's cheat codes thread.

This repository provides a clean, organized `titles/` directory structure for Switch title IDs and their corresponding cheat files (`.txt`). It is used as a dependency for [ns-emu-cheats-downloader](https://github.com/exefer/ns-emu-cheats-downloader).


## Structure
```
titles/
└── [TITLE_ID]/
    └── cheats/
        ├── [BUILD_ID].txt
```

## Source
Cheat data sourced from [GBAtemp.net](https://gbatemp.net/threads/cheat-codes-ams-and-sx-os-add-and-request.520293/).
All rights belong to the original cheat authors; this repo is just a cleaned-up mirror for easier processing by code.
